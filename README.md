# F00sel

F00sel is a Wayland-native application launcher, similar to rofi's
_drun_ mode.


## Screenshot

![Screenshot](doc/screenshot.png)


## Features:

- Wayland native
- Rofi drun-like mode of operation
- Emacs key bindings
- Icons!


## Limitations:

- No themes (but you **can** configure font and colors)
- No custom selection lists
